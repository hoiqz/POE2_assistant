## Context Reset Pattern

When you start a new task:

```
New Claude Conversation:

You: I'm building a POE2 companion app. Here's task [X]:

[Paste ONLY that task from TASKS.md]

Please help me build this. Here's the tech stack:
- Frontend: React + TypeScript + Tailwind
- Backend: Express.js + PostgreSQL
- AI: Claude SDK

Claude: Got it! Let's build [Task X]...
```

Claude can see the whole task and has all context it needs.
